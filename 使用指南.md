# Simulink 文档生成器 - 使用指南

## 🚀 快速开始

### 方式一：直接运行（需要 Python）

1. **安装 Python 3.8+**
   - 下载: https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. **安装依赖**
   ```bash
   pip install python-docx lxml
   ```

3. **运行程序**
   
   **GUI 模式**（图形界面）:
   ```bash
   python main.py
   ```
   
   **命令行模式**:
   ```bash
   python main.py model.slx
   python main.py model.slx output.docx
   ```

### 方式二：使用打包的 exe（无需 Python）

1. 运行 `build_windows.bat` 打包
2. 在 `dist` 目录找到 `Simulink文档生成器.exe`
3. 双击运行即可

---

## 📖 功能说明

### 支持的文件格式
- `.slx` - Simulink 模型文件（推荐）
- `.mdl` - 旧版 Simulink 模型文件（部分支持）

### 生成的文档内容
1. **模型概述** - 名称、版本、作者、统计信息
2. **系统架构** - 层级结构、子系统清单
3. **子系统详细说明** - 功能描述、端口、内部模块
4. **附录** - 数据字典（所有端口汇总）

### 智能功能描述
自动根据以下信息推断子系统功能：
- 子系统名称关键词（如 ADC、CAN、Input、Output 等）
- 内部模块类型（如 Gain、Sum、Logic 等）
- 端口信息（输入/输出数量）

---

## ⚙️ 命令行参数

```
用法: python main.py <slx_file> [output_file]

参数:
  slx_file     - Simulink 模型文件路径（必需）
  output_file  - 输出 Word 文档路径（可选，默认同目录）

示例:
  python main.py model.slx
  python main.py model.slx D:\docs\设计文档.docx
```

## 📋 GUI 使用步骤

1. 点击 **"浏览..."** 选择 `.slx` 文件
2. 选择输出目录（自动默认为模型所在目录）
3. 勾选需要的文档选项：
   - ✅ 包含信号流描述
   - ✅ 包含模块清单
   - ✅ 包含数据字典
4. 点击 **"解析模型"** 分析模型结构
5. 点击 **"生成文档"** 生成 Word 文档
6. 点击 **"预览信息"** 查看模型详细信息

## 🔧 常见问题

### Q: 提示"找不到模块 docx"
**A:** 运行 `pip install python-docx`

### Q: GUI 无法启动
**A:** 
- Windows: 确保 Python 安装时勾选了 tkinter
- macOS: 尝试 `brew install python-tk`

### Q: 解析结果显示模块数为 0
**A:** 
- 检查模型是否为空模型
- 确保模型已保存
- 尝试在 Simulink 中重新保存模型

### Q: 生成的文档中文乱码
**A:** 
- 确保系统安装了中文字体
- Word 中选择正确的编码打开

## 📦 打包成 exe

### Windows
```bash
# 方式一：运行批处理脚本
build_windows.bat

# 方式二：手动打包
pip install pyinstaller
pyinstaller --onefile --windowed --name "Simulink文档生成器" main.py
```

### macOS/Linux
```bash
chmod +x build_macos.sh
./build_macos.sh
```

打包后的文件在 `dist` 目录。

## 💡 技术说明

### 解析原理
`.slx` 文件本质是 ZIP 压缩包，内部包含：
- `simulink/blockdiagram.xml` - 模型结构定义
- `metadata/coreProperties.xml` - 元信息

解析器读取 XML，提取：
- System 节点 → 子系统
- Block 节点 → 模块
- P 节点 → 参数
- Port 节点 → 端口信息

### 限制
- ⚠️ 模块截图：无法自动生成
- ⚠️ 信号线追踪：部分支持
- ⚠️ Stateflow：可提取结构，无法还原图形
